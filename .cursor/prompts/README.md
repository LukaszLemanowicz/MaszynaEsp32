# Cursor Prompts

Ten folder zawiera zapisane prompty, które możesz szybko wywołać w Cursorze.

## Jak używać:

### Metoda 1: Przez Composer (@)
1. Otwórz Composer (Ctrl+I lub Cmd+I)
2. Wpisz `@` i zobaczysz listę dostępnych promptów
3. Wybierz prompt z listy lub wpisz jego nazwę

### Metoda 2: Przez Chat
1. W zwykłym czacie wpisz `@mvp-status` lub nazwę innego promptu
2. Cursor automatycznie załaduje zawartość promptu

## Dostępne prompty:

- `mvp-status` - Sprawdź status projektu używając 10x-mvp-tracker

## Dodawanie nowych promptów:

Po prostu dodaj nowy plik `.md` w tym folderze z treścią promptu. Nazwa pliku (bez rozszerzenia) będzie nazwą promptu.
